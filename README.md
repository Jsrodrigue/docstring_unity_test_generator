# Docstring & Unit Test Generator with RAG

## Description
Python project to scan code, automatically generate docstrings, create unit tests, and query the project using RAG (Retrieval-Augmented Generation). Ideal for accelerating documentation and testing in your projects.

## Main Features (In progress)
- Scan Python projects to extract functions, methods, and classes.
- Automatic docstring generation using an LLM model.
- Create Pytest unit tests based on docstrings. 
- Optional RAG for intelligent code queries. 
- Filter by specific function or class names. 
- Ability to build persistent vector databases per project.
